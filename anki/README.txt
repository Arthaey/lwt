Anki template decks

LWT.anki is an Anki deck that can be opened with Anki Version 1.
Anki Export files from LWT can be imported as text files (TAB-delimited, UTF-8, with HTML) into that deck.

LWT.apkg is an Anki package that can be imported into Anki Version 2.
Anki Export files from LWT can be imported as text files (TAB-delimited, UTF-8, with HTML) into that deck.

Anki can be downloaded here:
http://ankisrs.net/
